## 02 — BRAND PERSONA: ADMIN PB

Admin PB is the invisible character behind every post. Not a mascot, not a face — a tone. The most useful person in your pickleball WhatsApp group: the one who screenshots tournament posters, drops reg links before anyone asks, and reminds you the deadline is tomorrow.

### Admin PB is:
- **Pickleball player** - play social but join some tournaments
- **Helpful and fast** — drops the info before you ask
- **Slightly chaotic** — lowercase, rhetorical questions, light sarcasm
- **Very Malaysian** — natural lah, ah, jom, can or not, no need that
- **Trustworthy** — checks every platform, verifies before posting
- **A player too** — knows DUPR, knows the difference between Open and Novice
- **Never cringe** — no forced slang, no Gen Z cosplay, no emoji spam

### Admin PB is not:
- A government body or formal association
- A corporate brand or official media outlet
- An influencer with a face
- Trying too hard to be cool

## 03 — VOICE & COPY RULES

### Every post must answer at least one of:
1. **What** tournament is happening?
2. **Who** is it for? (skill cap, format, age, gender)
3. **When** does registration close?
4. **Why** should players care? (prize, format, location, value)
5. **What** should they do next? (save, share, register, DM)

### Caption structure
```
[Hook — one line, lowercase OK, scroll-stop required]
[Useful details — platform, deadlines, prize]
[Admin note — italic, optional, adds personality]
[CTA — save / share / link in bio]
```

### Language rules
- English first. Always.
- Malay as natural flavour — "jom daftar", "lah", "can or not" — never full Malay sentences in captions.
- Lowercase captions OK. Title-case slide headlines.
- No emoji spam. One emoji per line max, used like punctuation.
- No "please note that", "we are pleased to announce", "premier destination."
- Fragments are fine. Complete sentences optional.
- Player jargon is welcome: DUPR, dink, SLATE, Novice, Open.

### Do / Don't

| ✅ DO | ❌ DON'T |
|---|---|
| "12 tournaments. 7 states. Admin checked." | "We are pleased to announce 12 tournaments." |
| "Closing tonight 🚨" | "Reminder: registration deadline approaching." |
| "KL is the centre of Asian pickleball right now." | "MyTournament.PB is excited to share..." |
| "6 team slots left. Go now." | "Please register soon as slots are limited." |
| Lowercase fragments, cheeky | Formal complete sentences, corporate |

### Core lines (rotate)
- Every tournament. One place.
- We stalk tournaments so you don't have to.
- Admin checked. You scroll.
- Don't say nobody told you.
- Can join ah?
- Jom daftar.
- Save this. Thank admin later.
- Admin is tracking all of it.

### Never say publicly
- "Malaysia's #1"
- "ESPN of Malaysian pickleball"
- "Premier destination"
- "Comprehensive calendar"

## 04 — CONTENT FRANCHISES

Eight repeatable content types. Each has a fixed format, fixed day, fixed visual treatment.

### 1. Tournament Drop ← FLAGSHIP
**Every Friday. Carousel 6–7 slides.**
The primary weekly post. One THE PICK, supporting features, full list with platform pills, June teaser in end-of-month posts.

Slide flow:
1. Cover — hook (Court Green)
2. THE PICK deep-dive — full details (Court Green)
3. Watch/Register — actionable info (Deep Court or Court Green)
4. Full list — all open tournaments this weekend, platform pills (Off White)
5. June/upcoming teaser (Court Green)
6. CTA — Save · Share · Follow (Off White)

### 2. Closing Soon
**Tuesday or Thursday when deadline ≤72 hours.**
Urgency post. Rally Amber background. Big countdown. "Don't say nobody told you" sign-off.

### 3. Can Join Ah?
**Tuesday. Carousel.**
Picks one tournament, answers: who can enter, skill cap, fee, what to bring. Gateway content for beginners.

### 4. Worth It or Not?
**Tuesday. Carousel or Reel.**
One tournament breakdown — entry fee vs prize pool honest take. Editorial credibility builder.

### 5. Partner Needed Board
**Wednesday. Story or single post.**
Community utility. Players DM admin to get listed. Anonymous first-name format.

### 6. Scene Check
**Ad hoc — after major events. Reel or carousel.**
Post-event recap, cultural pulse of the Malaysian PB scene.

### 7. Admin Notes
**Monday. Story.**
Behind-the-scenes transparency: "found 3 new tournaments today, dropping Friday."

### 8. PB Wrapped
**Last day of each month. Carousel.**
Stats, top picks, best value, most expensive, biggest prize pool. Highly shareable.

### Weekly posting rhythm

| Day | Action |
|---|---|
| Mon | Data sweep (Sportssync + Baseline + SWP) · Admin Notes story |
| Tue | Can Join Ah? or Worth It or Not? · Closing Soon if urgent |
| Wed | Partner Needed Board story |
| Thu | Data sweep · Closing Soon if deadline tomorrow |
| **Fri** | **Tournament Drop carousel — flagship** |
| Sat/Sun | Engage comments · reshare stories · analytics check |

## 05 — VISUAL SYSTEM

### Color palette

| Name | Hex | Usage |
|---|---|---|
| **Court Green** | `#1A6B4A` | Primary — all carousel covers, headers, buttons |
| **Rally Amber** | `#F0A500` | Accent ONLY — logo ball, THE PICK tag, slide numbers, urgency strips, sticker pills |
| **Deep Court** | `#0D1F1A` | Dark bg — urgency posts, feature slides, closing soon |
| **Mint Wash** | `#E8F5EF` | Light card surfaces, dividers |
| **Off White** | `#F5F5F0` | Data slides, list slides, page backgrounds |
| **Mid Green** | `#6BAF8C` | Secondary text, admin notes, muted labels, footer handles |

**Amber rules:**
- Appears maximum once per slide (excluding logo ball)
- Never used as body text
- Never as a full background except on Closing Soon slides
- Used for: slide number "01", THE PICK tag, CLOSING SOON tag, FEATURED tag, logo ball

**Cover rule:** All carousel cover slides = Court Green. Deep Court only for urgency/Closing Soon posts.

### Typography

| Use | Font | Weight |
|---|---|---|
| Slide hooks, hero stats | Sora | 800 ExtraBold |
| Section headings | Sora | 700 Bold |
| Body, details, captions | DM Sans | 400–600 |
| Admin notes | DM Sans Italic | 400–500 |
| Slide number "01" | Sora | 800 |

**Sora:** headings and English display text only.
**DM Sans:** all body text, Malay text, captions, labels.
**Never mix more than these two fonts on a slide.**

### Carousel slide chrome

Every slide has:
- **Slide number** top-left: amber `01`, Sora Bold 54px
- **Section label** beside number: all-caps letter-spaced DM Sans 22px, colour varies by bg
- **Footer** centred: `@mytournament.pb`, DM Sans 24px

**Canvas:** 1080×1350px portrait PNG
**Padding:** 80px outer margin on all sides
**Upload:** Phone only — web upload crops 1080×1350 to square

### Gen Z layout devices

**A. Bold hook slides**
First slide is always a hook. Big Sora 800, often 130–150px. Single thought. Examples:
- "May's not done."
- "KL is the centre of pickleball in Asia."
- "Road trip. This weekend."

**B. Sticker tags / pills**
Small amber rounded rectangles, top-left. Letter-spaced uppercase DM Sans Bold 22px.
- `THE PICK` · `FEATURED` · `CLOSING SOON` · `LIVE NOW` · `NEW`

**C. Admin notes**
Italic Mid Green DM Sans bottom of slide, prefixed `admin note:`
- *admin note: reg is closed. but you can still watch.*
- *admin note: 6 slots left. that's it.*

**D. Urgency labels**
Amber strip or amber pill. White text. Reserved for Closing Soon posts.

**E. Platform source pills**
On every tournament list slide, each row shows a colour-coded pill identifying where to register:

| Platform | Pill background | Text color |
|---|---|---|
| Sportssync | `#E8F0FE` | `#1A3A8A` |
| Sports We Play | `#FEF3E8` | `#8A4A00` |
| Baseline | `#E8F5E9` | `#1A6B4A` |
| PPA Tour Asia | `#FFF3CD` | `#8A6000` |

**F. Save/share CTAs**
Last slide of every carousel:
- "Save this. Thank admin later."
- "Share with your doubles partner."
- "Follow + notifications on. Next drop Friday."

### What to avoid

| ❌ Drop | ✅ Use instead |
|---|---|
| Drop shadows, gradients | Flat color, sharp edges |
| Stock photos of generic athletes | Pure typography + color |
| Multiple fonts per slide | Sora + DM Sans only |
| Centered everything | Left-align body, center hero stats only |
| Emoji clusters | One emoji per line max |
| "Swipe →" arrows | Trust the audience |
| Decorative court line illustrations | Court Mark logo only |

## 07 — MULTI-CHANNEL ROLLOUT

Every Friday carousel post must include all of these:

### Instagram
- Upload all slides in order **from phone** (web crops canvas)
- Caption: hook → details → admin note → CTA + hashtags
- First comment: tag relevant organizers
- Core hashtags: `#pickleballmalaysia #pickleballmy #malaysiapickleball #klpickleball #pickleballasia` + event-specific

### Instagram Stories (3 stories same day)
- **Story 1** (immediately after posting): Reshare cover slide. Text sticker: "new drop 👆". Link sticker to bio.
- **Story 2** (same day, evening): Reshare the most urgent/useful slide. Poll or question sticker.
- **Story 3** (Tue–Thu following week): Reshare urgency slide if deadline approaching. "Closes [day]" text.

### Threads
- Shorter and more conversational than IG caption
- Lead with the hook, not the details
- Add context, not a copy-paste
- 3–5 lines max
- Tag @ppatourasia or organiser handles if relevant

### Facebook
- Full carousel (multi-image post, up to 10 photos)
- Longer caption OK — FB audience skews older, likes more context
- Bullet-point tournament list with emoji icons
- Pin to top of page until next Friday post

### Linktree (keep simple until website launches)
```
Latest post →
Sportssync →
Baseline →
Sports We Play →
DM on IG →
```

- [ ] Re-check ⚠ flagged rows for missing prize/fee data
- [ ] Confirm any deadlines closing this weekend
- [ ] Post Closing Soon story if any deadline within 48 hours

### Tracker rules
- Status = Verified before any public mention

- Prize Pool: RM number only. Use Prize Pool Note for "US$50K" or "TBC" situations
- Platform pill abbreviations: Sportssync · Baseline · Sports We Play · PPA Tour Asia
- Sort: chronological by Start Date always
